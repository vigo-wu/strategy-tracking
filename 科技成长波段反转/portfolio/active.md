## 操作备忘

| 动作 | 做法 |
| :--- | :--- |
| 新入选 | 追加一行；状态默认 `候选`，确认建仓后改 `持仓` 并填仓位% |
| 硬止损 / 破 MA10 / 满 4 周 / 候选失效 | **删除本表对应行**，并在当日日志「剔除清单」记录原因 |
| 止盈减仓 50% | 状态改 `减仓半仓`，仓位% 减半；若清仓则删行 |
| 自动选股 | 在主题目录执行 `python scripts/run_screener.py` → `output/screener_YYYY-MM-DD.json` |
| 历史回测 | `python scripts/run_backtest.py` → `output/backtest_*`；日线缓存在仓库根 `data/daily/` |
