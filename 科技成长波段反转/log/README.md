# 科技成长波段 · 日度观察日志

基于 [`model.md`](../model.md) 的交易日滚动记录。  
**更新频率**：每个交易日 1 次（建议尾盘公式确认后填写；或次日开盘前补记上一交易日）。

活页观察池：[`portfolio/active.md`](../portfolio/active.md)。  
Skill：仓库根 `.cursor/skills/a-share-daily-model/`。

## 使用方式

1. 复制 `_template.md` → 重命名为 `YYYY-MM-DD.md`（例：`2026-07-21.md`）
2. **先跟踪**：按 R1–R5 复核并更新 `portfolio/active.md`（不符合则移除）
3. **后选股**：运行通达信/同花顺公式，四维全过且催化核对后入库
4. 填写日日志各节，在下方「历史索引」追加一行
5. （可选）同步 `model.md`「日度滚动工作流」摘要

## 每日顺序（强制）

```
打开 active.md → 剔除触发 R1/R3/R4/R5 的标的
    → 运行选股公式 → 新票入库（候选/持仓）
    → 写 YYYY-MM-DD.md → 更新本索引
```

## 入选门槛（摘要）

须**同时**满足 `model.md` 第二节四维：

1. **基础过滤**：科技行业/300·688 优先、市值 50–500 亿、非 ST、上市满 1 年  
2. **空间与筹码**：3 个月高点回撤 ≥25%；近 5 日至少 3 日地量（&lt;60 日均量 40%）  
3. **动能反转**：MACD 零轴下金叉或底背离动能转向  
4. **起爆确认**：一阳穿三线 + 涨幅 ≥4.5% + 量能较前日放大约 2.5 倍且大于 5 日均量  

可建仓级还需：催化剂双向核对（事件驱动优先，无催化则降仓或仅观察）。

## 剔除门槛（摘要）

| 规则 | 条件 | 动作 |
| :--- | :--- | :--- |
| **R1 硬止损** | 收盘跌破起爆大阳线开盘价（或约定最低价） | 立即移出观察池 |
| **R2 止盈** | 约 1 周、浮盈 ≥15%、触 BOLL 上轨且换手 &gt;20% | 减仓 50% 或清仓移出 |
| **R3 破 MA10** | 趋势跟随段收盘跌破 MA10 | 余仓移出 |
| **R4 时间清仓** | 持股满第 4 周尾盘 | 全额移出 |
| **R5 候选失效** | `候选` 状态且不再满足入选硬条件 | 移出 |

**纪律**：已剔除标的禁止滞留 `portfolio/active.md`；只在当日日志「剔除清单」留痕。
