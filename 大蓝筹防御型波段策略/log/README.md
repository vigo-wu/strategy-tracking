# 大蓝筹防御 · 日度观察日志

基于 [`model.md`](../model.md) 的交易日滚动记录。  
**更新频率**：每个交易日 1 次（建议 **14:50 后**确认公式再写；或次日开盘前补记）。

活页观察池：[`portfolio/active.md`](../portfolio/active.md)。  
Skill：仓库根 `.cursor/skills/a-share-daily-model/`。

## 使用方式

1. 复制 `_template.md` → `YYYY-MM-DD.md`
2. **先跟踪**：按 R1–R5 复核并更新 `portfolio/active.md`
3. **后选股**：通达信公式或 `python scripts/run_screener.py`
4. 填写日日志，在下方「历史索引」追加一行
5. （可选）同步 `model.md`「日度滚动工作流」摘要

## 每日顺序（强制）

```
打开 active.md → 剔除/保本上移
    → 尾盘确认公式 → 新票入库（单票 ≤25% 仓）
    → 写 YYYY-MM-DD.md → 更新本索引
```

## 入选门槛（摘要）

须**同时**满足 `model.md` 第二节：

1. **基础蓝筹池**：沪深300或中证红利成分；市值 ≥500 亿；近 20 日均成交额 ≥3 亿；近 3 年分红、股息率 ≥3.5%  
2. **长期趋势**：收盘 > MA250，且 MA120 > MA250  
3. **箱体下轨**：收盘在 MA120 上方 3% 内；最低价触及 BOLL 下轨  
4. **右侧企稳**：CCI 自 -100 下方向上拐头（公式）；或手册所述 KDJ 低位金叉  

建仓：尾盘确认后 **2.5 成** 一次到位。

## 剔除 / 风控门槛（摘要）

| 规则 | 条件 | 动作 |
| :--- | :--- | :--- |
| **R1 硬止损** | 收盘跌破 **MA120×0.98** | 立即清仓移出 |
| **R2 保本上移** | 自买入反弹浮盈 ≥4%（或触 BOLL 中轨） | 止损改至成本；状态 `保本持仓` |
| **R3 止盈** | 触 BOLL 上轨，或浮盈 **8%–12%** | 全额清仓移出 |
| **R4 时间清仓** | 持股满 **第 3 周**尾盘 | 全额移出 |
| **R5 候选失效** | `候选` 且不再满足入选硬条件 | 移出 |

**纪律**：已剔除禁止滞留 `portfolio/active.md`；总回撤控制依赖「单票 ≤25% × 个股止损约 4–5%」。
