# 红利防御 · 日度观察日志

基于 [`model.md`](../model.md) 的交易日滚动记录。  
**更新频率**：每个交易日 1 次（建议 **14:50 后**确认公式再写；或次日开盘前补记）。

活页观察池：[`portfolio/active.md`](../portfolio/active.md)。  
Skill：仓库根 `.cursor/skills/a-share-daily-model/`。

## 使用方式

1. 复制 `_template.md` → `YYYY-MM-DD.md`
2. **先跟踪**：按 R1–R5 复核并更新 `portfolio/active.md`
3. **后选股**：通达信公式或 `python scripts/run_screener.py`
4. 填写日日志，在下方「历史索引」追加一行
5. （可选）同步 `model.md`「日度滚动工作流」摘要

## 每日顺序（强制）

```
打开 active.md → 剔除/保本上移
    → 尾盘确认公式 → 新票入库（单票 ≤25% 仓）
    → 写 YYYY-MM-DD.md → 更新本索引
```

## 入选门槛（摘要）

须**同时**满足 `model.md` 第二节：

1. **基础红利池**：中证红利成分或近 3 年平均股息 ≥4.5%；剔除 PE 历史 80% 分位以上；脚本近似市值 ≥300 亿、成交额 ≥1.5 亿  
2. **长期趋势**：收盘 > MA250，且 MA250 ≥ 昨 MA250（年线平/向上）  
3. **筹码拥挤度**：近 3 日均换手 ≤0.6%；收盘在 MA120 或 MA250 的 3% 内  
4. **右侧企稳**：站上 MA5，或 KDJ 低位金叉（K&lt;30）  

建仓：尾盘确认后 **2.5 成** 一次到位。

## 剔除 / 风控门槛（摘要）

| 规则 | 条件 | 动作 |
| :--- | :--- | :--- |
| **R1 硬止损** | 收盘跌破入选锚均线（MA120 或 MA250）**×0.975** | 立即清仓移出 |
| **R2 保本上移** | 自买入反弹浮盈 ≥3% | 止损改至成本；状态 `保本持仓` |
| **R3 止盈** | 浮盈 **6%–8%**，或 **RSI(6)>75** | 全额清仓移出 |
| **R4 时间清仓** | 持股满 **第 4 周**（约 20 交易日）尾盘 | 全额移出 |
| **R5 候选失效** | `候选` 且不再满足入选硬条件 | 移出 |

**纪律**：已剔除禁止滞留 `portfolio/active.md`；总回撤控制依赖「单票 ≤25% × 个股止损约 5%」。
