## 操作备忘

| 动作 | 做法 |
| :--- | :--- |
| 新入选 | 追加一行；默认 `候选`；尾盘确认后改 `持仓`，仓位填 **25** |
| 锚均线 | 入选时登记支撑侧（MA120 或 MA250，以 NEAR_LINE 命中者为准） |
| 防守止损价 | **锚均线×0.975**（下方 2.5%；R1 看收盘是否跌破） |
| R2 保本 | 浮盈 ≥3%：状态改 `保本持仓`，止损改买入价（成本） |
| R1 / R3 / R4 / R5 | **删行**，当日日志「剔除清单」留痕 |
| 自动选股 | `python scripts/run_screener.py` → `output/screener_*.json` |
| 历史回测 | `python scripts/run_backtest.py`；日线缓存仓库根 `data/daily/` |
